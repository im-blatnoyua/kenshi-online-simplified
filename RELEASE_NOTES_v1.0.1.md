# Kenshi Online v1.0.1 - Улучшенный инсталлятор

## 🔧 Исправления

### Улучшенный поиск Kenshi
- ✅ Добавлено больше путей автопоиска (Steam на дисках D:, E:, GOG, разные папки)
- ✅ Автоматический поиск через `find` в Linux (включая Flatpak Steam)
- ✅ Поддержка WSL путей (/mnt/c/)

### Улучшенный ввод пути
- ✅ Цикл повторного ввода при неверном пути (не нужно перезапускать)
- ✅ Автоматическое удаление кавычек из введенного пути
- ✅ Примеры правильных путей для пользователя
- ✅ Более понятные сообщения об ошибках

## 📦 Установка

Та же, что и в v1.0.0:

### Windows
```
1. Распакуйте архив
2. Запустите INSTALL.bat от имени администратора
3. Следуйте инструкциям
```

### Linux
```bash
tar -xzf kenshi-online-v1.0.1.tar.gz
cd release-package
chmod +x install.sh
./install.sh
```

## 🐛 Что исправлено

- Инсталлятор теперь находит Kenshi в большем количестве мест
- Если путь не найден автоматически, можно повторно вводить путь без перезапуска
- Убраны проблемы с кавычками в путях Windows
