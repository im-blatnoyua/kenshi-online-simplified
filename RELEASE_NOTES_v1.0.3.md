## 📦 Что включено

### Два инсталлятора на выбор:

**INSTALL.bat (рекомендуется)**
- Английский интерфейс
- Работает на всех версиях Windows (включая старые)
- Совместим с любыми настройками кодировки

**INSTALL_RU.bat**
- Русский интерфейс
- Требует поддержку UTF-8
- Используйте если первый показывает странные символы

### Linux:
- `install.sh` - работает на всех дистрибутивах с Wine/Proton

## 🚀 Установка

### Windows
1. Распакуйте архив
2. ПКМ на `INSTALL.bat` → Запуск от имени администратора
3. Следуйте инструкциям
4. Запускайте через ярлык "Kenshi Online"

### Linux
```bash
tar -xzf kenshi-online-v1.0.3.tar.gz
cd release-package
chmod +x install.sh
./install.sh
```
