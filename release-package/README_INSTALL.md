# Kenshi Online v1.0.4 - Simplified Installation

**16-player co-op multiplayer mod for Kenshi**

This is a simplified, ready-to-install version of Kenshi Online with automatic installers for Windows and Linux.

## Quick Install

### Windows
1. Download the latest release
2. Extract the archive
3. Run `INSTALL.bat` as Administrator
4. Launch via "Kenshi Online" desktop shortcut

### Linux (Wine/Proton)
1. Download the latest release
2. Extract and run:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
3. Launch via `~/kenshi-online.sh`

## Features

- Up to 16 players on a single server
- Dedicated server with persistence
- Master server with centralized browser
- Full network replication
- Native MyGUI HUD
- Just launch and play
