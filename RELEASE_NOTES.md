# Kenshi Online v1.0.0 - Simplified Release

## 🎮 Что нового

Это упрощенная версия Kenshi Online с автоматическими инсталляторами для максимально простой установки.

### Основные возможности

- ✅ **Автоматическая установка** - просто запустите INSTALL.bat (Windows) или install.sh (Linux)
- ✅ **До 16 игроков** на одном сервере
- ✅ **Выделенный сервер** с сохранением прогресса
- ✅ **Полная синхронизация** - персонажи, NPC, бои, строительство
- ✅ **Нативный интерфейс** - встроенный чат, список игроков, HUD
- ✅ **Готовые бинарники** - не требуется компиляция

### Установка

#### Windows
1. Скачайте `kenshi-online-simplified.zip`
2. Распакуйте в любую папку
3. Запустите `INSTALL.bat` от имени администратора
4. Играйте через ярлык "Kenshi Online"

#### Linux (Wine/Proton)
1. Скачайте `kenshi-online-simplified.zip`
2. Распакуйте и выполните:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
3. Запускайте через `~/kenshi-online.sh`

### Системные требования

- Kenshi (Steam или GOG версия)
- Windows 7+ или Linux с Wine/Proton
- Открытый порт 27800 UDP (для хостинга сервера)

### Известные проблемы

- Первый запуск может занять больше времени
- Некоторые антивирусы могут ложно срабатывать на инжектор
