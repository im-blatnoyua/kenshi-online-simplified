# Kenshi Online v1.0.4 - Исправление кодировки Windows-1251

## 🔧 Критические исправления

### INSTALL_RU.bat теперь работает на всех Windows!
- ✅ **Конвертирован в Windows-1251** (CP1251) - родная кодировка для русской Windows
- ✅ **Убрана команда `chcp 65001`** - больше не требуется UTF-8
- ✅ Исправлены ошибки: `'тей' is not recognized`, `'шибка' is not recognized`
- ✅ Теперь работает на Windows XP, 7, 8, 10, 11 без дополнительных настроек

## 📦 Что включено

### Два инсталлятора на выбор:

**INSTALL.bat** (английский)
- ASCII кодировка
- Универсальная совместимость
- Рекомендуется для международных систем

**INSTALL_RU.bat** (русский) - ИСПРАВЛЕН!
- Windows-1251 кодировка
- Работает на всех русских Windows
- Больше никаких проблем с кириллицей

### Linux:
- `install.sh` - работает на всех дистрибутивах с Wine/Proton

## 🚀 Установка

### Windows
1. Распакуйте архив
2. ПКМ на `INSTALL_RU.bat` → Запуск от имени администратора
3. Следуйте инструкциям
4. Запускайте через ярлык "Kenshi Online"

### Linux
```bash
tar -xzf kenshi-online-v1.0.4.tar.gz
cd release-package
chmod +x install.sh
./install.sh
```
